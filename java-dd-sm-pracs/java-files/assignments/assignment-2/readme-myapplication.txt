1. extract MyApplication.
2. open terminal here.
3. uncomment required import statement in Demo.java
4. uncomment required code portion in main method.
5. cd MyApplication
6. run "javac -d . *.java". it will create all required packages.
7. run "java Demo <command_line_argument_list>"
--- to run Triangle, it needs one arg(side of the triangle).
--- to run Rectangle, it needs two args(length and width of rectangle). 
--- to run TwoDEquation, it needs 3 args(degree of power 2(a), degree of power 1(b), constant(c) for a*x^2 + b * x + c)
